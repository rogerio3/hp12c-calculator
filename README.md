# HP12C Calculator

This is a simple implementation of an HP12C calculator. The HP12C is a financial calculator used for various calculations in finance and business.

## Features
- Basic arithmetic operations
- Financial calculations (NPV, IRR, etc.)

## Usage
To use the calculator, run the main module and follow the prompts to perform calculations.

## Installation
Clone the repository to your local machine:

```bash
git clone https://github.com/rogerio3/hp12c-calculator.git
```
